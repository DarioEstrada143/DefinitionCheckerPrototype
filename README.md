# DefinitionCheckerPrototype
We will be asking AI, specifically the AI Model Claude, to attempt to muster up a code that will be able to go through a map definition file for the video game Hearts of Iron IV, and attempt to isolate any potential issues that could cause the game to crash.

## First Prompt

**How familar are you Hearts of Iron IV modding? Specifically map modding?**

## Reflection

First off, I decided to start off my interaction with this simple little question, even though it might sound a bit silly because obviously the AI can just go search through the internet quickly and get itself familiarized, but that is the point. I wanted to get it familiarized with what I wanted to talk about before anything else. 

This was certainly a fascinating little experience to experiment with GenAI in this way, because for the longest time I have just been manually coding for Hearts of Iron IV mod development, and also attempting to just throw stuff like this into AI, stuff enormous like a definition file, and have it find issues. Having it actually create a code that can help me do exactly this seems like it has a lot of potential to do other things in HOI4 modding in order to speed up the process and deliever results, which is something very much needed when staying on top of a mod on the Steam Workshop page. SO it has a plethora of benefits there to speed things up look for errors that would otherwise take hours. 

However, one significant drawback from this is that, while yes I don't think anyone wants to go searching through a document with 7000+ potential provinces for errors, it mostly comes down to the fact that it could lead into some things like, not being that actually skilled with coding in HOI4 and being super relient on the AI to do the work for you. This can be a problem, because sometimes you really need to specifically prompt these bots in order to get the result you want, and if you don't even know what the issue is, especially in a code for something that you created for your mod for something like a nation that the bot doesn't really have a direct reference to on the internet, it could definitely get some things wrong with some corrections involving unique country_flags and how they interact with other files. Essentially, becoming too relient on the AI could create some buggy issues later down the line or even potentially complicate code more than it needs to be complicated.

But overall? Something like this has significant potential to help people like myself who are pretty much handling a solo mod project for HOI4.
